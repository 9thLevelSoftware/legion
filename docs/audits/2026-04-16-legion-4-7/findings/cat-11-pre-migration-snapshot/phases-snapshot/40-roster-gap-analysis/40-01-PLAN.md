**Section 2: Gap Analysis Algorithm**

Document the step-by-step algorithm:

1. **Load Configuration**
   - Parse roster-gap-config.yaml
   - Extract agent_limit, role_categories, severity thresholds

2. **Parse Agent Registry**
   - Load CATALOG.md
   - Extract all agents with their divisions and task types
   - Count total agents (verify against limit)
   - Build agent capability index

3. **Validate Intent Teams**
   - Load intent-teams.yaml
   - For each referenced agent, verify existence in CATALOG
   - Flag orphaned references (agents referenced but not in roster)
   - Document missing agents with reference locations

4. **Map Agents to Production Roles**
   - For each production role in config:
     - Find agents with matching task types
     - Score coverage: full (agent fully covers role), partial (some aspects), minimal (tangential)
     - Calculate coverage percentage
   - Build coverage matrix

5. **Identify Gaps**
   - For each under-covered role:
     - Calculate gap_score = 1.0 - coverage_percentage
     - Classify severity based on role priority and gap_score
     - Identify specific missing capabilities

6. **Check 52-Agent Limit**
   - Compare current_count vs agent_limit
   - If over limit: flag as constraint violation
   - Suggest replacement candidates from config

7. **Generate Report**
   - Summarize findings by category
   - Prioritize gaps by severity
   - Provide agent creation recommendations
   - Include limit compliance status
